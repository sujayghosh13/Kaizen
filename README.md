# Kaizen Calendar

A modern, interactive calendar application built with **React** and **Vite**. Kaizen helps you organize tasks, manage your schedule, and stay productive with a beautiful, themeable interface.

## Features

- **Multiple Views** — Day, Week, Month, and Year views with smooth transitions
- **Drag & Swipe Navigation** — Navigate dates by dragging or swiping across the calendar
- **Scroll Wheel Support** — Use your mouse wheel or trackpad to browse dates
- **Task Management** — Create, edit, and delete tasks with categories (Personal, Work, Health, Learning)
- **Smart Decompose** — AI-powered task breakdown using OpenRouter API
- **10+ Themes** — Dark mode, Tokyo Night, Neon Glow, Retrowave, and more
- **Browser Notifications** — Get alerts when tasks are due
- **Holiday Display** — Shows Indian holidays on the calendar
- **Mini Calendar Sidebar** — Quick date navigation with category filtering

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

The app runs at **http://localhost:3000** by default.

## Tech Stack

- **React 18** — UI framework
- **Vite** — Build tool & dev server
- **Vanilla CSS** — Custom design system with CSS variables
- **OpenRouter API** — Smart task decomposition (optional, requires API key)
